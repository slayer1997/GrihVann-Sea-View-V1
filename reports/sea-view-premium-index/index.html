The Sea-View Premium Index

What a 180° arc is worth over a 90° glimpse — measured, not asserted.
Home  /  Sea-View Premium Index

Last updated: July 2026

Everyone in Mumbai pays for “sea-facing.” Almost no one prices the difference between a full 180° arc and a 90° sliver. This index does.

The idea

A “sea view” is not one thing. Two flats in the same building, both marketed sea-facing, can offer wildly different views: one sweeps a continuous 180° arc from horizon to landmark, the other catches a 90° wedge between two neighbouring towers. The Sea-View Premium Index isolates that single variable — the width and clarity of the arc — and asks what it is worth in rupees per square foot, holding location, floor and building quality constant.

Why it is different

Standard property indices track price by locality or building. None of them separate the view arc from everything else, because doing so needs per-unit obstruction data that brokers do not publish. GrihVann already records that data for every listing — the exact arc, the obstruction risk and the orientation, floor by floor. That makes this the one index that can actually be built from primary observation rather than inference.

The arc bands

We classify every unit into one of four arc bands. The premium multipliers shown are an Illustrative framework only — placeholders that show how the index will read once GrihVann’s transacted data is applied. They are not measured values.

Illustrative framework only · not measured. The index base (1.00) is the full unobstructed arc; other bands express the fraction of that value a narrower arc commands. Verified multipliers replace these once transacted pairs are analysed.

Method

  • Matched pairs. Compare units in the same building and floor band that differ only in arc, to strip out location and height.
  • Arc measurement. Use GrihVann’s per-unit obstruction report to assign each unit an arc band in degrees.
  • Transacted price. Anchor to registered transaction values, not asking prices, so the premium reflects what buyers actually paid.
  • Index. Express every band as a fraction of the full-arc price to produce a single, comparable multiplier per locality.

Why it matters for buyers

If the index holds, it means a buyer can be paying a full-arc price for a partial-arc view — or, better, can find a genuinely full-arc unit priced as if it were merely “sea-facing.” Making the premium explicit is the whole point: it turns a vague label into a number you can negotiate against.
